# MAO クイックスタート 🚀

## 1分で始める

### ステップ1: プロジェクト初期化
```bash
cd your-project
mao init
```

### ステップ2: タスクを指定して起動
```bash
# シンプルな例
mao start "README.mdを作成して"

# 開発タスク
mao start "login.pyのユニットテストを追加"

# 複雑なタスク（複数エージェント）
mao start "完全な認証システムを実装"
```

**これだけ！** エージェントが自動的に起動してタスクを実行します。

## 画面の見方

### ダッシュボード（自動で起動）
```
┌─ Multi-Agent Orchestrator ────────────┐
│ エージェントの状態とログが表示されます   │
│                                       │
│ キー操作:                             │
│   q = 終了                            │
│   r = 更新                            │
│   Tab = フォーカス移動                │
└───────────────────────────────────────┘
```

### tmuxグリッド
別ターミナルで以下を実行：
```bash
tmux attach -t mao
```

3x3のグリッドで各エージェントのログが見えます：
```
┌─ 📋 MANAGER ──┬─ 🔧 WORKER-1 ─┬─ 🔧 WORKER-2 ─┐
│ ログ...       │ ログ...       │ ログ...       │
├───────────────┼───────────────┼───────────────┤
│ 🔧 WORKER-3   │ 🔧 WORKER-4   │ 🔧 WORKER-5   │
│ ログ...       │ ログ...       │ ログ...       │
├───────────────┼───────────────┼───────────────┤
│ 🔧 WORKER-6   │ 🔧 WORKER-7   │ 🔧 WORKER-8   │
│ ログ...       │ ログ...       │ ログ...       │
└───────────────┴───────────────┴───────────────┘
```

**tmux操作:**
- `Ctrl+B` → `d` : デタッチ（元の画面に戻る）
- `Ctrl+B` → `矢印` : ペイン移動
- `Ctrl+B` → `z` : ペイン拡大/縮小

## よくある質問

### Q: エージェントが何も実行しない
A: タスクを指定してください：
```bash
mao start "タスクの内容"
```

### Q: tmuxが邪魔
A: `--no-tmux`で無効化：
```bash
mao start --no-tmux "タスク"
```

### Q: どのロールを使えばいい？
A: 通常は指定不要（自動選択）。特定のロールを使いたい場合：
```bash
mao roles  # ロール一覧を確認
mao start --role coder_backend "APIを実装"
```

### Q: APIキーの設定は？
A: 環境変数に設定：
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```
