# ダッシュボード再設計

## 現在の問題点

1. **複雑すぎる**: 多くのウィジェットが表示されて何を見れば良いかわからない
2. **見た目が悪い**: レイアウトが煩雑
3. **使い方がわからない**: どう操作すれば良いかわからない

## 本当に必要な機能

### 最小限の機能

1. **エージェント状態** - 今何が動いているか
2. **ログ表示** - エージェントが何をしているか
3. **タスク情報** - 今何をやっているか

### あれば良い機能

4. タスク進捗バー
5. トークン使用量/コスト
6. エラー表示

## 推奨: レイアウト1（シンプル）

**理由:**
- 必要な情報が見やすい
- エージェントの状態が一目でわかる
- ログが見やすい
- 操作がシンプル

**実装方針:**
1. 上部: タスク情報（固定）
2. 中部: エージェント一覧（スクロール可能）
3. 下部: 選択したエージェントのログ（リアルタイム更新）

**カラースキーム:**
- ✓ 完了: 緑
- ⚙ 実行中: 黄色
- ⏸ 待機中: グレー
- ✗ エラー: 赤

## 削除する機能

1. 承認パネル（複雑すぎる）
2. 詳細なメトリクス（基本情報のみ）
3. タスクコントロールパネル（シンプル化）
4. 最近の活動ウィジェット（ログで十分）

## キーボード操作

```
q     - 終了
r     - 画面更新
↑↓   - エージェント選択
Enter - エージェント詳細
←→   - ログのエージェント切り替え
```
