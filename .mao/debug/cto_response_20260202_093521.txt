**テスト概要:**
- 5つの異なるロール（researcher, coder_backend, tester, reviewer, planner）のエージェントを同時起動
- 各エージェントに異なるモデル（Haiku, Sonnet, Opus）を割り当て
- 並行処理による効率的な実行
- 詳細なログとサマリー出力

**スクリプトの特徴:**
1. API Keyが未設定の場合はモックモードで動作
2. 各エージェントの実行状況を個別にトラッキング
3. 実行時間、トークン使用量、エラーハンドリングを含む詳細レポート
4. ログファイルを個別に生成

実際のAPI呼び出しでテストするには、環境変数 `ANTHROPIC_API_KEY` を設定してください。
